🚨 Dépannage

Si des pages ne fonctionnent pas :

  1. Vérifiez que les serveurs sont démarrés :
    • Frontend : cd apps/frontend && npm run dev
    • Backend : cd apps/backend && npm run dev
  2. Vérifiez les ports :
    • Frontend : http://localhost:3000
    • Backend : http://localhost:3001
  3. Vérifiez la base de données PostgreSQL
